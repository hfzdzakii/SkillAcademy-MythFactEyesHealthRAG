Read me first
Hello, I'm Haztsu. I'm the creator of this website. This project was made to fulfill the graduation requirements for the Artificial Intelligence Skill Academy Pro Bootcamp at Ruang Guru.

All answers provided by this website are 100% AI-generated. Therefore, please be cautious when accepting any information obtained here. As the creator, I'm aware of my limited knowledge in health. Additionally, I don't have a real expert to verify all the information regarding myths, facts, or trivia about eyes health. Because of this, there's a possibility the AI may provide irrelevant answers.

Please use this AI responsibly! If you have any vision-related concerns, it's best to consult an eye doctor directly.
What technologies behind this project?
This is Langchain-based project using RAG (Retrieval-Augmented Generation). Langchain's component used are:
  • Document Loader
  • Vector Stores
    • Vector Store used in this project is Chroma
  • Text Splitter
  • Prompt Template
    • Used as both input and instructions for the AI
  • LLM Model
    • Gemini-2.0-flash-lite-001 is being used for this project
How to gather data about eyes health?
Please pay close attention again! Due to the creator's limited knowledge in health, the creator built the dataset independently by prompting popular free AI tools like OpenAI ChatGPT and Google Gemini
How to verify the information?
The creator performed validation using popular and free AI tools like OpenAI ChatGPT and Google Gemini. For example, if both AIs agreed that a piece of eye health information was factual, it was included in the dataset. However, if the AIs provided conflicting answers, the related information was discarded.
